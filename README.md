# Atlas Dev Lab Setup

Setup automatizado para transformar Ubuntu em um laboratório de desenvolvimento:

- Java 21 (SDKMAN)
- Node LTS (NVM) + Angular + Next.js
- Android SDK (CLI)
- Podman (rootless)
- VS Code + extensões
- IntelliJ IDEA Community
- Git + GitHub CLI
- AWS CLI

## Uso

```bash
chmod +x setup-devlab.sh
./setup-devlab.sh
```
